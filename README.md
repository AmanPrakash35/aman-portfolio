# Professional Portfolio Website

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS. This portfolio template is designed for IT professionals and can be easily customized with your own information.

## Features

- **Modern Design**: Clean, professional layout with smooth animations
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Sticky navigation with smooth scrolling
- **Interactive Components**: Hover effects, mobile menu, and animated elements
- **SEO Friendly**: Proper meta tags and semantic HTML
- **Fast Performance**: Built with Vite for optimal loading speeds

## Sections Included

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Section**: Personal information and quick facts
3. **Skills Section**: Categorized skills and expertise display
4. **Experience Section**: Work history with achievements
5. **Projects Section**: Featured projects with technology tags
6. **Contact Section**: Contact form and social media links
7. **Footer**: Professional footer with social links

## Customization Guide

### Personal Information

Update the following in `src/App.jsx`:

- **Name**: Replace "Aman" with your name (appears in navigation, hero, and footer)
- **Title**: Update "IT Professional" with your job title
- **Location**: Replace "Your Location" with your city/country
- **Email**: Update "your.email@example.com" with your email
- **Phone**: Update the phone number in the contact section
- **Social Links**: Replace GitHub and LinkedIn URLs with your profiles

### About Section

Update the about text and quick facts:
- Experience years
- Number of projects completed
- Technologies count
- Certifications count

### Skills Section

Modify the skills arrays in the Skills section to reflect your actual skills:
- Programming Languages
- Frameworks & Tools
- Soft Skills

### Experience Section

Update the work experience array with your actual job history:
- Job title
- Company name
- Employment period
- Job description
- Key achievements

### Projects Section

Replace the placeholder projects with your actual projects:
- Project title
- Description
- Technologies used
- Project link

### Contact Section

Update contact information and social media links in the contact section.

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Building for Production

To create an optimized production build:

```bash
npm run build
```

The built files will be in the `dist` directory.
